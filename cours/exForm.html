Sur cette page, vous pouvez envoyer un fichier HTML qui sera alors stocké sur ce site. Pour cela, il faut indiquer votre prénom, puis choisir le fichier de votre ordinateur à envoyer. Seuls les fichiers HTML seront acceptés.
Vous pouvez envoyer autant de fichiers que vous le voulez (indiquez votre prénom à chaque fois), mais attention, vous ne pourrez pas les supprimer.
